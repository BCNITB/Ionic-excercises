Exercicis amb Ionic

Pàgina web per a crear una aplicació amb el framework Ionic que sigui accessible pels lectors de pantalla. Aquesta aplicació s'ha realitzat amb fins educatius.

Contingut

  • Exercici 1: creació de l'entorn de treball i inici de l'aplicació.
  • Exercici 2: Afegint un segon nombre aleatòri.
  • Exercici 3: Missatge d'error.
  • Exercici 4: Inserint una animació.
  • Exercici 5: Llançant un efecte sonor.
  • Exercici 6: Creant un comptador d'intents.
  • Exercici 7: Mostrant una classe o una altre.
  • Exercici 8: Limitant els intents.
  • Exercici 9: Creació botó.
  • Apèndix: Afegint un tercer valor.
  • Apèndix: Modificant la suma per la multiplicació
